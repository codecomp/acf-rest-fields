Add ACF fields to existing REST endpoints instead of using of custom endpoints and add custom endpoints for ACF only content.

== Description ==



== Installation ==

1. Upload plugin directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Functionality Roadmap ==

* Add custom field content to posts and pages endpoints.
* Add custom field content to all custom post type queries.
* Add custom field content to taxonomy queries
* Add custom fields to Attachments, Widgets, Comments, Users
* Find solution to adding ACF fields to menu queries in other plugins
* Add Options page queries.
* Add ability to request specific fields only.
* Add ability to request no ACF content
* Add toggle to all ACF fields in back end to blacklist them from the REST requests
